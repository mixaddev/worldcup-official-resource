# worldcup-official-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源形式存储，不针对任何特定域名或网站。

## 项目简介

这是一个面向通用场景的 HTML 页面归档仓库。仓库内容以独立页面为主，可用于演示、存档、离线浏览或作为其他项目的参考资源。

## 目录说明

```
worldcup-official-resource/
├── pages/              # 存放所有独立 HTML 页面
├── assets/             # 页面引用的静态资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/` 目录下每个子文件夹代表一个独立页面，包含对应的 HTML 文件及其依赖。
- `assets/` 目录存放公共资源，供多个页面共享使用。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开运行。
- 页面之间不存在相互依赖，每个页面均可独立访问。
- 归档内容可能包含示例、演示或历史版本，不保证所有页面均处于活跃维护状态。
